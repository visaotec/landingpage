Sistema Assistencia Tecnica

O Sistema MasterTec foi desenvolvido pensando no mercado de Assistência Técnica, com ele você pode controlar o fluxo de manutenção dos equipamentos. O Sistema trás os seguintes módulos: Cadastro de clientes, fornecedores, equipamentos, marcas, categoria, subcategoria, produtos, serviços, termo de garantia . Com o Sistema MasterTec Oficina é possível abrir ordem de serviço, controlar o status , o cliente pode acompanhar o andamento da OS pelo painel do cliente. Para agilizar o atendimento o Sistema conta com Assinatura Eletrônica na OS para Clientes e Técnicos.

Sistema Delivery

Nesse Sistema o cliente através de um link / site conseguirá de forma simples efetuar seu pedido, sem ter que fazer cadastro ou login, somente pelo número do telefone estará relacionado ao pedido, vamos ter o painel administrativo para os cadastros e para gestão dos pedidos, teremos relatórios, impressão de comprovante e pedidos para cozinha em impressora térmica e muito mais.
